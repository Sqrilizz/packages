### Installed Packages Export

### A simple Python script that exports all installed packages from your system.

It generates two files:

installed_packages.json — structured list

installed_packages_list.txt — plain text list

Usage
python3 packages.py


The output files will appear in the same folder.

```installed_packages_list.txt prewiew```
```
[ SYSTEM ]
 - xorg
 - gdm3
 - vulkan-tools

[ DEVELOPMENT ]
 - python3
 - build-essential
 - git

[ GAMING ]
 - steam
 - wine
 - lutris

...
```
